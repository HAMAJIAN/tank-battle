# 坦克大战 (Tank Battle)

一个用纯 HTML5 Canvas 实现的经典坦克大战游戏。

## 玩法

- **WASD / 方向键** — 移动坦克
- **空格 / J** — 发射子弹
- 消灭红色敌方坦克获得分数
- 子弹可以摧毁砖墙，开辟新的路径
- 每关消灭足够敌人即可过关

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (无框架)
- Web Audio API 音效

## 运行

直接在浏览器中打开 `index.html` 即可。
